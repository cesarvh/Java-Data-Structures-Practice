<h1> About: </h1> 
Implementations of various Data Structures implemented whilst Lab Assisting for CS61B, Spring 2016. Made from scratch in Python, C and Java

- LinkedListDeque: Circular Linked List implementation of a Double-Ended Queue 
- ArrayDeque: Circular Array List implementation of a Double-Ended Queue
- CommitTree: Directed Graph, mimicking the structure of Git
- BST: Binary Search Tree
